# Regeln

Die Mitglieder auf diesem Server und die Mitglieder der zugeordneten Server müssen dem folgen Nutzungsbedingungen von Discord (dis.gd/terms) und die Discord-Community-Richtlinien (dis.gd/guidelines). Jeder Verstoß gegen eine dieser Bestimmungen führt zum sofortigen und dauerhaften Ausschluss. In einigen Fällen bedeutet dies auch eine potenzielle Meldung an Discord.

Obwohl der Server keine zusätzlichen Regeln hat, sollten Mitglieder ihren gesunden Menschenverstand walten lassen, und jedes Verhalten, das die Moderatoren für unangemessen halten, kann zu einer Strafe führen.

**a.** Die Server Eidgenossenschaft hat 4 offizielle Sprachen: Englisch, Deutsch, Italienisch und Spanisch.
Dies sind die einzigen Sprachen, die für Chats auf dem Server zu Moderationszwecken akzeptiert werden. Wenn ein Server jedoch die Voraussetzungen erfüllt, um Mitglied der Eidgenossenschaft zu sein, schließt die Verwendung oder Nichtverwendung der offiziellen Sprachen ihn nicht aus.

**b.** Der Beitrittsprozess zur Server Eidgenossenschaft muss nach dem vorgegebenen Muster durchgeführt werden. Entspricht es nicht den Standards, wird der Antrag abgelehnt.

**c.** Der Mitgliedsantrag kann aus folgenden Gründen abgelehnt werden:

- Der Server entspricht nicht den Nutzungsbedingungen oder den Community-Richtlinien.
- Der Server hat weniger als 20 Mitglieder.
- Der Server hat weniger als 12 % aktive Mitglieder.
- Der Server hat Sicherheitsfehler oder ist nicht gut konfiguriert.
- Der Server war Teil der Server Eidgenossenschaft und wurde aus einem anderen Grund ausgeschlossen.

**d.** Der Bundesrat ist das oberste Verwaltungsorgan des Bundes. Seine Aufgabe ist es, die Mitgliedschaftsanträge zu prüfen und abzustimmen, ob sie angenommen werden oder nicht. Seine Mitglieder werden monatlich in Wahlen gewählt, die auf dem Server der Server Eidgenossenschaft oder, wenn es bequemer ist, auf dem Mitgliedsserver mit den meisten Benutzern stattfinden.

**e.** Das Mandat jedes Rates dauert einen Monat. Wenn beispielsweise Ende Dezember ein neuer Rat gewählt wird, tritt dieser am 1. Januar sein Amt an und geht am 1. Februar in den nächsten gewählten Rat über.

**f.** Die Mitglieder des Bundesrates können sich ohne Amtszeitbeschränkung wiederwählen, jedoch nicht nacheinander, daher muss es mindestens einen Rat geben, dem sie nicht angehören, bevor sie wieder regieren können.

**g.** Die Wahlen zum Rat, die im folgenden Monat ihr Amt antreten, finden am 15. eines jeden Monats statt, sodass Benutzer, die bis zum 10. erscheinen möchten, ihre Kandidatur für die Wahlen bekannt geben möchten. Bis heute wird ihnen ein Kanal freigeschaltet, auf dem sie ihre Kandidatur und ihre Vorschläge präsentieren können.
